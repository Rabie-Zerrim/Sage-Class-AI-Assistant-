# Project Description: Class AI Assistant (Sage)

## Overview
The Teacher Bot Assistant is a web-based application featuring Sage (Class Assistant AI), an AI-powered chatbot designed to help teachers with educational tasks. The application provides a comprehensive suite of tools for student management, lesson planning, content creation, and resource recommendations.

## Main Purpose and Functionality
The primary purpose is to assist educators in creating personalized, effective teaching strategies through AI-driven recommendations. Sage analyzes student learning styles (visual, auditory, kinesthetic) and provides tailored lesson plans, teaching methods, and educational resources including videos and slides.

## Key Features
- **AI Chat Interface**: Interactive conversation with Sage for teaching assistance and queries
- **Class Chat Rooms**: Dedicated chat interfaces for each class where students can collaborate and communicate together
- **Student Management**: Add and manage student profiles with learning styles, ages, classes, and interests
- **Lesson Plan Creation**: Generate personalized lesson plans based on student data and topics
- **Post Creation**: Create and organize educational posts by topic with filtering and search
- **Topic Browsing**: Explore subjects with associated lessons and teachers
- **Resource Recommendations**: Access curated videos and lecture slides for various subjects
- **Multi-step Conversations**: Guided workflows for complex tasks like lesson planning

## Technology Stack
- **Backend**: Python 3.8+ with Flask/FastAPI for AI processing
- **AI/ML Frameworks**: TensorFlow 2.0+, scikit-learn 0.24+, OpenAI API integration
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Data Persistence**: Browser localStorage for client-side data, PostgreSQL/SQLite for server-side
- **Styling**: Custom CSS with responsive design and animations
- **Architecture**: Full-stack application with RESTful API and modular components

## Primary Use Cases
- Teachers seeking personalized teaching strategies for individual students
- Educators creating lesson plans tailored to different learning styles
- Instructors managing student information and tracking progress
- Teachers accessing educational resources and multimedia content
- Faculty members collaborating through shared posts and discussions
- Educational professionals getting AI-powered suggestions for classroom activities
